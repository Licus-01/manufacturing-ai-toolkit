# Manufacturing AI Toolkit 🏭

An AI-assisted documentation system designed to streamline quality management workflows in manufacturing environments — freeing engineers from repetitive paperwork so they can focus on innovation.

## 🎯 Purpose

Engineers in manufacturing SMEs spend a significant portion of their time on documentation tasks — SOPs, audit reports, corrective actions, and compliance submissions. This toolkit leverages AI to automate and assist these processes, targeting an estimated **50–70% reduction** in documentation time.

## ✨ Features

- **SOP Generation** – Draft Standard Operating Procedures from process inputs with consistent structure and formatting
- **Audit & Compliance Documents** – Generate CQI assessment summaries, corrective action reports (CARs), and IATF 16949-related documentation
- **Supplier Documentation** – Assist with PPAP/APQP documents, process flowcharts, and Customer Specific Requirements (CSR) reviews
- **Bilingual Support** – Full Chinese–English translation with manufacturing and automotive industry terminology
- **Process Flowcharts** – Auto-generate visual process flows from structured input

## 🏗️ Target Users

- Quality engineers and supplier quality teams in automotive manufacturing
- SME suppliers managing customer-facing compliance documentation
- Teams working under IATF 16949, PPAP, APQP, or CQI frameworks

## 🔧 Tech Stack

- Claude API (Anthropic) – Core language and reasoning engine
- Python – Backend processing
- To be expanded as development progresses

## 💡 Why This Exists

Most AI tools are designed for general office use and lack the domain knowledge required for manufacturing quality systems. Enterprise-grade solutions exist but are out of reach for smaller suppliers. This toolkit targets that underserved gap — practical, affordable AI tooling for the shop floor and the quality office.
